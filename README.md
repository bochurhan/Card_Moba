# Card_Moba - 3v3同步回合制卡牌MOBA

## 项目概述
3v3同步回合制卡牌MOBA游戏，核心玩法：1+2固定分路、瞬策-定策单窗口双轨体系、中枢塔PVE+BOSS战、分路崩塌3v3决胜、25回合硬封顶轻量化对局。

## 技术栈
- **客户端**：Unity 2022.3 LTS + C#
- **后台**：ASP.NET Core + SignalR + C#
- **共享结算库**：.NET Standard 2.1 C# 类库
- **数据存储**：MySQL 8.0 + Redis 7.0
- **通信协议**：Protobuf + WebSocket
- **辅助工具**：.NET Core 控制台工具

## 项目结构
```
Card_Moba/
├── Client/              # Unity客户端项目
├── Server/              # ASP.NET Core后台项目
├── Shared/              # 前后端共享代码（结算库、协议、配置模型）
├── Tools/               # 辅助工具（配置表转换、校验）
├── Config/              # 游戏配置表（Excel源文件 + 导出JSON）
├── Docs/                # 项目文档
└── Card_Moba游戏技术文档.md  # 技术选型文档
```

## 开发规范
- 命名规范：帕斯卡命名法（类、方法、公共字段）+ 驼峰命名法（私有字段、局部变量）
- 注释规范：每个类、方法必须有XML注释
- 方法规范：单方法代码行数不超过100行
- 提交规范：按功能模块提交，提交信息需清晰描述变更内容
