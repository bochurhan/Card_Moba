# 3v3同步回合制卡牌MOBA 核心设计文档
**文档版本**：V1.6 中枢塔规则优化版
**更新日期**：2026年02月22日
**文档状态**：核心规则已全部确认，可用于落地开发
**核心定位**：市面独家轻量化同步回合制卡牌MOBA，彻底剥离传统MOBA操作门槛，将胜负核心100%聚焦于「局前卡组构筑、回合内卡牌博弈、团队战术配合」，兼顾卡牌高频操作爽感与纯策略博弈公平性，单局时长严格锁定8-15分钟。

---

## 一、底层不可突破铁律（无任何例外）
1.  单局胜负中，卡组构筑、资源规划、长线决策的策略权重≥80%，手部操作/反应速度占比严格≤20%；
2.  完整保留TCG卡牌核心灵魂，绝不做成技能图标换皮的传统MOBA；
3.  所有对抗类操作同步预提交、统一优先级结算，无先后手差异、无手速比拼空间；
4.  正常对局无自由移动权限，跨路交互仅可通过远程施法类卡牌实现；
5.  单局总回合数硬封顶25回合，绝对禁止无限制膀胱局；
6.  瞬策牌核心作用于自身卡组循环，定策牌覆盖所有对抗类效果，边界零模糊。

## 二、对局整体节奏框架
固定闭环节奏，具体触发回合数归入数值模块后续专项敲定：
| 对局阶段 | 核心内容 |
|----------|----------|
| 匹配准备阶段 | 玩家提前锁定「1人solo位/2人双人组」位置，完成局前卡组构筑，系统按「1solo+2双人组」固定阵容匹配 |
| 第一分路对线阶段 | 1v1+2v2固定分路运营，无自由移动，仅可触发死亡支援移动，完成基础卡组循环铺垫 |
| 第一中枢塔发育回合 | 固定节点触发，全员强制进入中枢塔，分「小怪清剿个人发育→全局BOSS阵营对抗」两阶段完成，单回合内结束 |
| 第二分路对线阶段 | 延续固定对线规则，解锁中枢无限制攻击权限，对局节奏逐步加快 |
| 第二中枢塔发育回合 | 固定节点触发，高阶小怪清剿+终极BOSS决战，完成卡组最终优化，为决胜阶段铺垫 |
| 分路崩塌决胜阶段 | 第二中枢塔回合结束后强制触发，分路合并为全局决战主路，进入全员3v3终极对抗 |
| 兜底强制结算 | 25回合未分胜负，按固定优先级判定胜负，无平局 |

## 三、分路与移动核心规则（最终确认）
### 3.1 分路基础规则
- 匹配前锁定位置，对局开局后玩家默认进入对应分路，solo路全程限1名同队玩家，双人路全程限2名同队玩家；
- 开局后可队内协商换路，需互换位置的双方玩家均同意，触发固定惩罚：换路双方下回合能量上限-1；
- 正常对局状态下，无任何自由跨路移动、中枢往返移动权限，跨路交互仅可通过远程施法类定策牌实现。

### 3.2 唯一移动权限：队友死亡支援
- 触发时机：同队玩家死亡的回合结算完成后，死亡玩家的复活等待回合内可提交支援指令；
- 权限限制：单次死亡事件最多仅可1名存活玩家触发支援移动，无任何移动代价，无需跳过下回合；
- 归位规则：死亡玩家复活生效的同时，支援玩家自动回归原锁定分路，死亡玩家回归自身原锁定分路，无需额外操作、无额外代价。

### 3.3 分路崩塌决胜规则
- 触发节点：第二中枢塔发育回合结束后的首个对抗回合，两条分路强制合并为全局决战主路；
- 决胜规则：仅可通过决战主路攻击敌方中枢，关闭分路偷家路径，所有同路联动规则对全队生效。

## 四、单回合核心流程（单窗口双轨体系）
单回合基准时长约25s，全程双方完全同步，无先后手差异：
| 阶段序号 | 阶段名称 | 核心规则 |
|----------|----------|----------|
| 1 | 回合开始结算期（全自动执行） | 同步执行固定抽牌、能量回满、持续buff/debuff生效，校验对局胜负、玩家死亡状态与支援移动权限 |
| 2 | 同步操作窗口期（唯一操作窗口，固定时长） | 【双方同步开启/关闭，操作互相隔离仅自身可见】<br>1. 瞬策牌：打出立即生效于自身牌库/手牌/能量池，操作不可逆，支持连锁刷牌；<br>2. 定策牌：可提交/取消/修改，仅锁定不生效，预扣对应能量，窗口期关闭后不可修改；<br>3. 可提交支援移动、能量转移指令，同路队友手牌/已提交指令队内全透明 |
| 3 | 指令最终锁定期 | 关闭操作窗口，锁定所有指令，非法指令直接作废 |
| 4 | 统一结算期 | 按固定优先级同步结算所有内容，无先后手差异：反制牌→防御牌→增益/减益牌→伤害牌→支援牌→传说特殊牌；瞬策牌仅结算限制级直伤效果 |
| 5 | 濒死判定期 | 统一执行死亡判定，作废死亡玩家未生效操作，校验对局胜负条件 |
| 6 | 回合结束期 | 强制弃光所有手牌，清理临时效果，更新对局状态，校验强制结算触发条件 |

## 五、卡牌体系核心规则
### 5.1 卡牌主分类与双轨归属
| 主分类 | 核心定义 | 双轨体系归属 |
|--------|----------|--------------|
| 伤害类 | 核心效果为对目标造成直接伤害，附加效果不改变主分类 | 全部归入定策牌，预提交后统一结算 |
| 功能类 | 核心效果为非直接伤害的对局功能，无任何直伤 | 仅卡组运转、纯自身资源类可归入瞬策牌，其余对抗类（含远程施法、控制、反制）全部归入定策牌 |
| 传说特殊牌 | 强效果+苛刻触发条件的单局上限牌，单卡组限带1张 | 全部归入定策牌，设置独立结算优先级 |

### 5.2 核心边界规则
- **瞬策牌**：仅可作用于自身，不可对敌方卡牌、场面、buff/debuff造成影响，仅开放固定小额直伤的限制级设计，具体限制由单卡定义；
- **定策牌**：覆盖所有对抗类效果，跨路交互仅可通过定策牌中的远程施法类卡牌实现，无其他跨路路径；
- **反制牌规则**：统一归为定策牌，必须提前预提交，触发条件明确，提交后给敌方强制预警（仅告知有反制，不披露具体类型），可反制瞬策/定策牌，不可反制其他反制牌；
- **卡组循环规则**：牌库抽空后弃牌堆立即洗回，支持无限循环；回合结束强制弃光手牌，仅带「手牌保留」词条的卡牌可例外；抽牌超出手牌上限则直接进入弃牌堆，无额外惩罚。

## 六、中枢塔发育回合 优化后终版规则
### 6.1 基础规则
1.  **触发机制**：对局固定节点触发，触发后暂停分路对抗，双方所有玩家强制进入中枢塔全局场景，无任何例外；
2.  **时长控制**：全程在**1个完整同步回合内完成**，不新增额外对局回合，不拉长整体对局总时长，复用常规回合的「瞬策-定策单窗口双轨体系」，玩家无需学习新规则；
3.  **核心定位**：给前期弱势角色提供稳定的发育窗口，通过PVE小怪战完成卡组优化、强度提升，再通过全局BOSS战拉开阵营差距/给劣势方翻盘机会；
4.  **阶段划分**：回合内分为两个完全同步的阶段，全程双方无先后手、无手速比拼，所有操作同步提交、同步结算。

### 6.2 第一阶段：小怪清剿（PVE个人发育）
#### 核心规则
1.  **挑战模式**：同阵营3名玩家各自独立挑战小怪，玩家之间、敌我阵营之间互不干扰，各自结算挑战结果；
2.  **难度选择**：小怪分为低/中/高3个难度档位，玩家可自主选择挑战档位，弱势角色可选择低难度拿保底收益，强势角色可选择高难度冲超额奖励；
3.  **操作规则**：复用常规回合的双轨体系，玩家在同步操作窗口期内提交瞬策/定策牌指令，窗口期关闭后统一结算伤害，判定挑战结果；
4.  **挑战限制**：本阶段仅可使用攻击/增益/自身卡组循环类卡牌，不可使用干扰敌方的卡牌，所有效果仅对PVE小怪生效。

#### 收益规则（保底+超额，无空车风险）
1.  **保底收益**：无论挑战是否成功，所有参与玩家均可获得**固定免费卡组优化权限**（1次免费删卡/卡牌升级）+ 基础金钱奖励，彻底保障前期弱势角色的发育需求；
2.  **挑战成功收益**：根据所选难度，额外获得对应奖励：
    - 低难度：额外1张随机职业卡牌；
    - 中难度：额外1张随机职业卡牌+小额金钱；
    - 高难度：额外个人遗物+大额金钱+2次免费卡组优化权限；
3.  **收益归属**：本阶段所有收益完全归玩家个人所有，个人遗物可自愿无偿分配给队内任意队友，无需投票确认。

#### 劣势方兜底规则
前序分路对抗阶段处于劣势的阵营，本阶段所有玩家对小怪的伤害获得固定增益（如20%），缩小双方发育差距，避免前期崩盘。

### 6.3 第二阶段：全局BOSS决战（3v3vE阵营对抗）
#### 核心规则
1.  **触发条件**：小怪清剿阶段结算完成后，自动进入本阶段，无需额外操作；
2.  **挑战模式**：双方阵营共同挑战同一个全局世界BOSS，比拼阵营总伤害，同时可通过通用卡牌互相干扰，全程使用常规对局的通用卡牌，无需专属回合卡牌；
3.  **操作规则**：开启第二个同步操作窗口期，玩家可提交两类定策指令，窗口期关闭后统一结算：
    - **BOSS攻击指令**：提交攻击/增益类卡牌，伤害计入己方阵营总伤害；
    - **阵营干扰指令**：提交功能类卡牌，实现降低敌方阵营总伤害、干扰敌方卡牌生效等效果，不可直接对敌方玩家造成伤害；
4.  **结算规则**：窗口期关闭后，同步结算双方的BOSS有效总伤害、干扰效果，总伤害更高的阵营获胜。

#### 收益规则
1.  **阵营胜负奖励**：总伤害更高的阵营，获得**全队共享BOSS遗物**优先选择权，BOSS遗物为全队生效，无队内分配矛盾；
2.  **个人保底奖励**：所有对BOSS造成伤害的玩家，无论阵营胜负，均可获得额外基础金钱奖励；
3.  **翻盘奖励**：前序对抗劣势的阵营，若本次BOSS战总伤害反超获胜，可额外获得全队免费卡组优化权限，放大翻盘可能性。

### 6.4 回合衔接规则
1.  中枢塔回合全流程结束后，所有玩家自动回到己方原锁定分路，血量、能量、持续buff全部重置为分路对抗回合初始状态；
2.  本回合获得的新卡牌、遗物、卡组优化权限，下一个分路对抗回合立即生效，可正常使用；
3.  本回合所有临时效果全部清空，仅保留永久成长奖励。

## 七、死亡与复活规则
1.  玩家死亡后进入复活等待回合，仅可提交「复活后回归原分路」的移动指令，不可使用卡牌、参与对局操作；
2.  复活生效时，支援玩家与死亡玩家均自动回归各自原锁定分路，本次复活移动无「跳过下回合」的刚性代价；
3.  玩家判定死亡后，本回合已提交、未生效的所有操作全部作废，已生效的持续效果全部清空。

## 八、胜负条件与时长兜底
### 胜利条件（无平局）
达成任一条件立即获胜：
1.  摧毁敌方中枢；
2.  团灭对方所有玩家；
3.  25回合强制结算后判定为优势方。

### 强制结算优先级
第25回合未分胜负，按以下优先级从高到低判定：
1.  敌方中枢已损失血量占比更高的一方获胜；
2.  单局总击杀数更高的一方获胜；
3.  单局对敌方玩家总伤害量更高的一方获胜；
4.  全队剩余总血量更高的一方获胜。

### 节奏推进机制
每5个回合，双方所有伤害类卡牌效果永久提升10%，鼓励主动进攻，避免消极拖局。
