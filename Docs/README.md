## 最近更新记录

### 2026-02-25 (V1.1)

**架构变更**：
- `CardSubType` 枚举已废弃，功能合并到 `CardTag` (Flags 枚举)
- `CardConfig.SubType` 属性已移除，改用 `Tags` 字段
- 结算优先级不再由标签决定，而是由 `CardEffect.EffectType` 数值范围驱动

**配置模板更新**：
- Excel 模板中的 `EffectId` 统一添加 `E` 前缀（如 `E1001-1`）以避免 Excel 日期格式问题
- 参见 `Config/Excel/Cards_Template*.csv` 和 `Config/Excel/Cards_Template.md`
