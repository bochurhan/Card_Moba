# 文档索引

## 当前重点

当前项目的主线是：

- 以 `Shared/BattleCore` 作为单场战斗内核
- 以 `Shared/MatchFlow` 承载整局 `4+1`
- 以 `Server` 托管权威对局
- 以客户端 `IBattleClientRuntime` 支撑本地与联机双模式 UI
