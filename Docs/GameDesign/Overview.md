# 核心玩法概述

**文档版本**：V1.0  
**适用对象**：需要快速理解游戏核心机制的所有人  
**阅读时间**：5 分钟

---

## 🎮 游戏定位

**3v3 同步回合制卡牌 MOBA**

| 特性 | 说明 |
|------|------|
| **核心体验** | 卡组构筑 + 回合博弈 + 团队配合 |
| **策略权重** | ≥80%（卡组构筑、资源规划、长线决策） |
| **操作权重** | ≤20%（无手速要求，无先后手差异） |
| **单局时长** | 8-15 分钟（25回合硬上限） |

---

## 🃏 双轨卡牌体系

游戏只有两种主分类的卡牌：

| 类型 | 打出时机 | 生效时机 | 核心作用 |
|------|----------|----------|----------|
| **瞬策牌** | 操作窗口期内随时打出 | 打出后**立即生效** | 卡组循环（抽牌、能量调整），可造成限制级直伤 |
| **定策牌** | 操作窗口期内预提交 | 回合末**统一结算** | 所有对抗效果（伤害、防御、控制、反制） |

**核心边界**：
- 瞬策牌：作用于**自身**（手牌、牌库、能量），不能跨路
- 定策牌：作用于**对抗**（敌方、场面），可带「跨路生效」标签

## ⏱️ 单回合流程（7阶段）

每回合按固定顺序执行，**不可逆**：

```
┌─────────────────────────────────────────────────────────────┐
│ 1. 回合开始结算期（自动）                                      │
│    → 抽牌、能量回满、Buff生效、反制牌校验                        │
├─────────────────────────────────────────────────────────────┤
│ 2. 同步操作窗口期（唯一操作窗口，约25秒）                        │
│    → 双方同时操作，互相不可见                                   │
│    → 瞬策牌：打出即生效                                        │
│    → 定策牌：提交/修改/取消，仅锁定不生效                        │
├─────────────────────────────────────────────────────────────┤
│ 3. 指令最终锁定期                                             │
│    → 关闭操作窗口，锁定所有定策牌指令                            │
├─────────────────────────────────────────────────────────────┤
│ 4. 定策牌统一结算期                                           │
│    → 按优先级同步结算所有定策牌                                 │
│    → 反制→防御→伤害→功能（无先后手）                            │
├─────────────────────────────────────────────────────────────┤
│ 5. 濒死判定期                                                 │
│    → 统一执行死亡判定，清空死亡玩家的效果                        │
├─────────────────────────────────────────────────────────────┤
│ 6. 回合结束期                                                 │
│    → 弃光手牌、清空能量、更新状态                               │
└─────────────────────────────────────────────────────────────┘
```

---

## 📊 定策牌结算优先级

回合末，所有定策牌按以下顺序**同步结算**（同层级无先后手）：

| 层级 | 类型 | 说明 |
|------|------|------|
| **Layer 0** | 反制效果 | 无效化/惩罚被反制的卡牌 |
| **Layer 1** | 防御/数值修正 | 护甲、护盾、攻击力增减 |
| **Layer 2** | 主动伤害 + 触发效果 | 伤害结算 → 触发式效果（反伤、吸血等） |
| **Layer 3** | 功能效果 | 控制、资源、传说牌等 |

## 🗺️ 对局流程总览

```
匹配准备 → 锁定位置（Solo/双人）、选择卡组

    ↓

第一分路对线期（2场战斗）
    → 1v1 Solo路 + 2v2 双人路，固定分路

    ↓

第一中枢塔发育战
    → 小怪清剿（个人PVE）→ BOSS讨伐（阵营对抗）

    ↓

第二分路对线期（2场战斗）
    → 继续分路对线，解锁更多权限

    ↓

第二中枢塔发育战
    → 高阶小怪 + 终极BOSS

    ↓

全局决战死斗期
    → 分路合并，3v3 终极对抗

    ↓

胜负判定
```

---

## 🏆 胜负条件

达成任一条件**立即获胜**：

1. 摧毁敌方中枢
2. 团灭对方所有玩家
3. 敌方投降
4. 25回合封顶后优势方胜出

**封顶判定优先级**：中枢损失比 → 总击杀数 → 总伤害量 → 剩余血量

## ⚔️ 分路与移动

| 机制 | 说明 |
|------|------|
| **固定分路** | Solo路1人，双人路2人，**无自由移动** |
| **永久换路** | 战斗间可申请，有冷却和能量惩罚 |
| **死亡支援** | 队友死亡后可临时支援，战斗结束自动归位 |
| **跨路卡牌** | 仅带「跨路生效」标签的定策牌可跨路 |
