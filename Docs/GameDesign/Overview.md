# 核心玩法概述

**文档版本**：V2.0  
**适用对象**：需要快速理解游戏核心机制的所有人  
**阅读时间**：5 分钟  
**最后更新**：2026年03月21日

---

## 🎮 游戏定位

**1v1 / 2v2 同步决策团队卡牌竞技**

> 首版以 **1v1** 先行验证核心机制，系统层面预留 **2v2** 扩展接口。

| 特性 | 说明 |
|------|------|
| **核心体验** | 同步决策博弈 + 卡组构筑 + 团队配合 |
| **策略权重** | ≥80%（卡组构筑、资源规划、读牌判断） |
| **操作权重** | ≤20%（无手速要求，无先后手差异） |
| **单局时长** | 预计 10–18 分钟（死斗阶段设有回合上限） |

## 🃏 双轨卡牌体系

游戏只有两种主分类的卡牌：

| 类型 | 打出时机 | 生效时机 | 核心作用 |
|------|----------|----------|----------|
| **瞬策牌** | 操作窗口期内随时打出 | 打出后**立即生效** | 资源控制、手牌调整、节奏铺垫；可对**中枢**造成直接伤害，但不直接伤害英雄 |
| **定策牌** | 操作窗口期内预提交 | 出牌阶段结束后**统一公开结算** | 主要对抗效果（伤害、防御、控制、反制、团队联动） |

**核心边界**：
- 瞬策牌：主要作用于**自身状态**（手牌、牌库、能量、节奏铺垫），可包含对中枢的直接伤害，原则上不对英雄造成即时伤害
- 定策牌：作用于**对抗场面**（敌方英雄、中枢、团队目标），可带「跨路生效」标签

## ⏱️ 单回合流程

每回合按固定顺序执行，**不可逆**：

```
┌─────────────────────────────────────────────────────────────┐
│ 1. 回合开始结算期（自动）                                      │
│    → 抽牌、能量回满、Buff生效、持续效果结算                      │
├─────────────────────────────────────────────────────────────┤
│ 2. 同步操作窗口期（唯一操作窗口，默认约20秒）                    │
│    → 双方同时操作，对手操作不可见                               │
│    → 瞬策牌：打出即生效                                        │
│    → 定策牌：提交/修改/取消，仅锁定不生效                        │
│    → 所有玩家锁定后可提前结束本阶段                              │
├─────────────────────────────────────────────────────────────┤
│ 3. 指令最终锁定期                                             │
│    → 关闭操作窗口，锁定所有定策牌指令                            │
├─────────────────────────────────────────────────────────────┤
│ 4. 定策牌统一公开与结算期                                      │
│    → 所有定策牌翻开，按优先级同步结算                            │
│    → 反制 → 防御/数值修正 → 伤害 → 功能（无先后手）              │
├─────────────────────────────────────────────────────────────┤
│ 5. 濒死判定期                                                 │
│    → 统一执行死亡判定，清空死亡玩家的效果                        │
├─────────────────────────────────────────────────────────────┤
│ 6. 回合结束期                                                 │
│    → 弃光手牌、清空能量、更新状态、事件进度结算                   │
├─────────────────────────────────────────────────────────────┤
│ 7. 信息回顾阶段                                               │
│    → 展示本轮关键结算摘要，帮助玩家理解发生了什么                  │
└─────────────────────────────────────────────────────────────┘
```

## 📊 定策牌结算优先级

回合末，所有定策牌按以下顺序**同步结算**（同层级无先后手）：

| 层级 | 类型 | 说明 |
|------|------|------|
| **Layer 0** | 反制效果 | 无效化/惩罚被反制的卡牌 |
| **Layer 1** | 防御/数值修正 | 护甲、护盾、攻击力增减 |
| **Layer 2** | 主动伤害 + 触发效果 | 伤害结算 → 触发式效果（反伤、吸血等） |
| **Layer 3** | 功能效果 | 控制、资源、传说牌等 |

## � 核心玩法循环

### 宏观循环（跨局）

1. 选择英雄（即职业）与预构筑卡组；
2. 进入对局，开局公开本局团队目标与奖励池；
3. 在每个决策窗口中抽牌、出牌、规划资源；
4. 通过**瞬策牌**调整节奏与手牌，通过**定策牌**进行隐藏部署；
5. 争夺阶段性团队目标或中盘事件奖励；
6. **每场战斗结束后进入局内构筑窗口**（拿卡/删卡/升级/回血），对本局 build 进行微调；
7. 通过压制、资源优势、关键爆发或摧毁核心目标获取胜利；
8. 对局结束后复盘，调整构筑与英雄搭配。

### 单回合微循环

1. **准备阶段**：抽牌、刷新资源、事件进度预告；
2. **同步出牌阶段**：所有玩家在限定时间内同时出牌；
3. **瞬策结算**：瞬策牌打出即生效，立即处理；
4. **定策公开**：出牌阶段结束，所有定策牌统一翻开；
5. **分桶结算**：按优先级依次结算（反制→防御→伤害→功能）；
6. **战场更新**：血量、护盾、状态、事件进度等统一更新；
7. **信息回顾**：展示本轮关键结算摘要。

## �🗺️ 对局流程总览

```
匹配准备 → 选择英雄（即职业）与卡组
             ↓
         开局公开本局团队目标与奖励池

             ↓

         第1场：对线建立（6回合）
             → 1v1 分路对抗，围绕团队目标进行早期布局

             ↓

         第2场：对线深化（6回合）
             → 继续分路，试探对手，低速推进团队目标进度

             ↓

         第3场：团队目标争夺期·激活（6回合）
             → 团队目标进入争夺期（具体激活时机视设计调整）
             → 双方仍在分路对抗，同时推进目标完成度

             ↓

         第4场：团队目标争夺期·兑现（6回合）
             → 争夺期延续，完成目标可获得奖励优先选择权
             → 死斗前完成奖励结算

             ↓

         第5场：死斗阶段（最多10回合）
             → 双路合流，2v2 汇集同一战场
             → 携带入场护盾，第3回合起持续失血收束

             ↓

         胜负判定
```

## 🏆 胜负条件

达成任一条件**立即获胜**：

1. **摧毁敌方中枢**（核心目标本体血条归零）
2. **死斗阶段团灭对手**（死斗阶段双方均到场后才启用）
3. **敌方投降**
4. **死斗回合上限到达后**，按剩余总生命值/中枢状态判定优势方胜出

> ⚠️ 死斗阶段之前的分路对线期，不以"团灭所有玩家"作为直接胜利条件；需要通过中枢伤害或等到死斗阶段来决出胜负。

**核心目标结构**：
- 每名玩家拥有一个核心目标（中枢）
- 中枢由**外层护盾/护甲** + **本体生命值**两层构成
- 本体生命值归零即触发获胜判定

## 🧙 英雄与职业

- 采用**英雄即职业**设计：每个英雄直接对应一种完整职业风格（类似《炉石传说》）
- 每个英雄拥有专属卡池与核心机制，风格差异鲜明
- 首版控制英雄数量，优先保证风格辨识度
- 每个英雄应有 1 个非常鲜明的核心机制，便于玩家快速理解自身在团队中的职责
- **不允许出现只能在死斗中发挥作用的"纯辅助型"英雄**：所有英雄都必须具备基本单线作战能力，职业只能有侧重，不能失去单线价值

## 🃏 卡组结构

首版采用**小卡组**结构，推荐默认 **16 张**：

| 构成 | 数量 | 说明 |
|------|------|------|
| 基础牌 | ~10 张 | 稳定通用行动（攻击、防御、过牌、资源调节等） |
| 核心牌 | 1 张 | 本局卡组关键风格支点 |
| 公共补充牌 | 2 张 | 通用策略与节奏适配 |
| 职业牌 | 3 张 | 强化英雄风格与辨识度 |

## 👥 队友信息公开度

- **对敌信息**：对手打出定策牌时，仅能看到已提交几张，默认不知道具体内容；结算期公开并结算
- **对友信息**：采用**高公开度**方案，队友手牌、待结算区、本轮意图原则上均可主动查看
- 设计意图：对敌隐藏博弈，对友透明协作，清晰区分"团队协作层"与"对手博弈层"
