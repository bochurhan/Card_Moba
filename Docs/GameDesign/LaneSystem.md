## 🎯 设计目标

| 目标 | 实现方式 |
|------|----------|
| **保留 MOBA 结构感** | 双路并行对抗，各路独立推进 |
| **控制复杂度** | 无显式站位、无自由移动，专注于出牌博弈 |
| **放大团队协作** | 跨路联动牌 + 死斗阶段合流 |
| **战场可解释** | 每名玩家当前回合的行动范围清晰明确 |

---

## 🗺️ 战场抽象

首版不采用传统 MOBA 式完整地图，也**不设显式站位系统**。  
战场感主要通过以下四项结构表达：

1. **各路核心目标的攻防**（玩家的"基地"）
2. **角色职责分工**（英雄即职业，每人职责明确）
3. **多场战斗组成的对局阶段**（5场结构）
4. **阶段性团队目标争夺事件**（中盘关键节奏点）

> 玩家不需要处理前后排、左右区、移动路径等空间问题，  
> 每回合只需关注：**本轮作用于谁、保护谁、压制谁、争夺什么目标**。

## �️ 双路结构（2v2 模式）

### 基础配置

```
┌──────────────────────────────────────────┐
│              A 阵营  vs  B 阵营           │
│                                          │
│  [上路]   A1  ──────────────  B1         │
│                                          │
│  [下路]   A2  ──────────────  B2         │
│                                          │
└──────────────────────────────────────────┘
```

- 两条分路**完全对称**，首版不区分"资源路""压力路"等功能差异
- 每名玩家操控 **1 个英雄**，绑定所在分路
- 绝大多数决策发生在**当前分路的 1v1 对线**中
- 少量跨路牌与死斗合流作为战略补充

### 首版（1v1 模式）

首版以 **1v1** 先行验证核心机制，双路结构为 2v2 扩展预留。  
1v1 时战场即为单路对抗，无分路概念。

## 🔒 分路规定

| 规则 | 说明 |
|------|------|
| **固定分路** | 开局锁定，整局主要在所属分路进行对抗 |
| **无自由移动** | 不存在"走位"或"跨路移动"操作 |
| **跨路影响** | 仅通过带「跨路生效」标签的定策牌实现有限跨路 |
| **死斗合流** | 进入死斗阶段后，双路自动合并为 2v2 同场战斗 |

### 为什么不设自由移动？

| 如果允许自由移动 | 固定分路的优势 |
|-----------------|---------------|
| 6人同时出牌，信息难以追踪 | 每人只需关注本路对手 |
| 全员抱团成为唯一最优解 | 分路各有独立博弈价值 |
| 增加操作门槛与反应要求 | 专注于读牌、预判与资源决策 |
| 复杂化结算逻辑 | 目标范围清晰，结算可解释 |

## 🎯 跨路牌

### 设计原则

- 跨路牌数量应**少而精**；
- 主要承担**保护与节奏支援**作用，暂不以跨路直接伤害为核心方向；
- 跨路联动是战术加分项，而非日常主循环；
- 仅**定策牌**可带「跨路生效」标签，瞬策牌不可跨路作用于英雄。

### 效果示例方向

| 效果类型 | 说明 |
|----------|------|
| 跨路支援护盾 | 为另一路队友提供护盾或防护加成 |
| 跨路能量补充 | 向队友补充少量资源，协助关键回合 |
| 跨路控制辅助 | 对另一路敌方施加轻微干扰（效果打折） |

## � 死斗阶段：双路合流

进入**第5场死斗阶段**后，双路合并为同一战场：

```
┌──────────────────────────────────────────┐
│              死斗阶段（2v2）              │
│                                          │
│     A1 + A2   ──────────   B1 + B2       │
│                                          │
│  · 跨路限制完全解除                       │
│  · 所有卡牌可选任意有效敌方目标            │
│  · 入场护盾生效（见下方）                  │
│  · 第3回合起持续失血开始                  │
└──────────────────────────────────────────┘
```

### 入场护盾

每名角色进入死斗时自动获得入场护盾，由两部分叠加：

| 来源 | 规则 |
|------|------|
| **推进折算护盾** | 此前对敌方核心造成的累计伤害 × 50% |
| **生命保底护盾** | 当前生命值的 20%–40%（暂定，待测试调整） |

> 设计意图：让前期推进优势自然继承到死斗，同时缓冲开场第一轮双人集火。

### 持续失血（强制收束）

| 回合 | 失血规则 |
|------|---------|
| 第 1–2 回合 | 正常展开，无失血 |
| 第 3 回合起 | 每回合开始时，所有存活角色直接失去生命值（逐回合递增） |

- 失血效果**直接作用于生命**，不先扣护盾
- 目标：死斗后续 7 回合内，持续失血累计约造成角色总生命值 **30%–40%** 的理论伤害
- 具体数值为可测试参数，首版测试后根据平均局长调整

### 死斗胜负判定

| 优先级 | 条件 | 结果 |
|--------|------|------|
| 1 | 某一方两名角色均被击败（团灭） | 立即结束，对方胜 |
| 2 | 达到 10 回合上限 | 比较双方总剩余生命值，多者胜 |

> 10 回合上限为首版暂定参数，后续根据测试调整。

## 📊 对局阶段与分路关系

| 阶段 | 战场形态 | 分路状态 |
|------|----------|----------|
| 第1场：对线建立（6回合） | 1v1 分路 | 各路独立 |
| 第2场：对线深化（6回合） | 1v1 分路 | 各路独立 |
| 第3场：团队目标争夺·激活（6回合） | 1v1 分路 | 各路独立，团队目标进入争夺期 |
| 第4场：团队目标争夺·兑现（6回合） | 1v1 分路 | 各路独立，完成争夺后奖励结算 |
| 第5场：死斗（最多10回合） | 2v2 合流 | 分路合并，全员同场 |

> ⚠️ 团队目标争夺期的具体激活时机可调整，不一定固定在第3场开始。
