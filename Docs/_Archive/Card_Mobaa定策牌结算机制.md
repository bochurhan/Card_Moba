## 一、全局不可突破底层铁律（无任何例外，永久不可修改）
本文档所有规则均严格遵循以下铁律，任何场景、任何卡牌设计均不得突破：
1.  **公平性铁律**：跨阵营/跨玩家的定策牌效果，永远同步结算，无任何先后手差异；仅同一名玩家自身提交的卡牌，按其最终锁定的提交顺序在同层级内依次结算，不涉及跨阵营博弈，无手速比拼空间。
2.  **结算顺序铁律**：定策牌核心结算顺序永久锁定为**反制结算→防御与数值修正→主动伤害与触发闭环→非依赖效果收尾**，依赖关系决定结算顺序，任何场景不得颠倒。
3.  **卡牌边界铁律**：严格遵守主文档定策牌核心定义——预提交锁定、窗口期关闭后统一结算、覆盖所有对抗类效果，不得与瞬策牌核心边界混淆。
4.  **连锁封顶铁律**：单回合内，仅玩家主动提交的定策伤害牌可触发1次连锁效果，所有连锁衍生的伤害/效果，绝对不再触发新的连锁，杜绝无限循环逻辑漏洞。
5.  **多子类型拆分铁律**：单张定策牌的多个独立效果，必须按效果依赖关系拆分到对应结算层级，禁止整卡强制归属某一子类型/单一层级；单效果被反制不影响同卡牌其他未被反制的效果正常结算。
6.  **伤害结算铁律**：所有主动定策伤害牌，必须在同一环节统一、一次性同步结算，绝对不允许跨阵营分次结算单张伤害牌，可出现双向击杀。
7.  **结算与表现解耦铁律**：所有定策牌结算逻辑必须在动画播放前完成全量预结算，锁定不可逆的最终结果；动画仅为表现层，绝对不影响、不修改最终结算结果。
8.  **信息安全铁律**：同步操作窗口期内，绝对不播放任何定策牌动画、不展示任何卡牌信息、不泄露任何操作相关的数值变化，杜绝信息提前泄露。

| 术语名称 | 官方定义 |

| 定策牌 | 主文档定义的两大核心卡牌类型之一，操作窗口期内预提交锁定、窗口期关闭后统一结算，覆盖所有对抗类效果，核心分为伤害、防御、反制、增益/减益、支援、传说特殊6个子类型 |
| 有效定策牌 | 未被反制、未被作废、符合生效条件的定策牌，是结算与触发效果的唯一统计基准 |
| 堆叠层 | 定策牌结算的核心执行单元，按效果依赖关系划分的固定结算层级，层级顺序永久不可颠倒 |

## 三、单回合定策牌全生命周期结算流程
本流程100%兼容主文档的单回合7步核心框架，仅细化定策牌相关环节，无任何冲突：
| 回合阶段序号 | 阶段名称 | 定策牌相关核心操作 | 无例外规则 |
|--------------|----------|---------------------|------------|
| 1 | 回合开始结算期（全自动执行） | 1. 校验上回合提交的跨回合定策牌（反制牌、持续效果）的触发条件<br>2. 结算上回合锁定的持续增益/减益效果<br>3. 校验玩家死亡状态、支援移动权限 | 本阶段仅校验触发条件，不执行新的定策牌结算，不可回溯上回合已完成的结算 |
| 2 | 同步操作窗口期（唯一操作窗口） | 1. 玩家可提交/修改/取消定策牌，系统预扣对应能量，仅锁定不生效<br>2. 同路队友可互相查看已提交的定策牌，敌方完全不可见<br>3. 能量不足无法提交定策牌，非法指令实时拦截 | 窗口期内定策牌仅锁定不生效，所有操作互相隔离，不对外泄露任何信息 |
| 3 | 瞬策牌实时结算环节 | 玩家打出的瞬策牌实时结算，仅作用于自身卡组/资源/状态，不影响已锁定的定策牌 | 瞬策牌仅可锁定定策牌相关的触发状态，不可修改、作废已提交的定策牌 |
| 4 | 指令最终锁定期 | 1. 关闭操作窗口，永久锁定所有定策牌指令，不可再修改/取消<br>2. 校验非法指令，作废不符合生效条件的定策牌，全额返还预扣能量<br>3. 后台启动全量预结算流程 | 锁定后定策牌指令不可逆，仅反制牌可使其作废，无其他修改路径 |
| 5 | 定策牌统一结算期（核心） | 后台按「同步堆叠式结算规则」完成全量预结算，锁定最终结果；前台按预结算结果播放对应动画 | 严格遵循堆叠层顺序结算，同层级内跨阵营效果完全同步结算 |
| 6 | 濒死判定期 | 1. 统一执行死亡判定，作废死亡玩家未生效的所有定策牌效果<br>2. 清空死亡玩家已生效的持续效果<br>3. 实时校验对局胜负条件 | 已完成结算的定策牌效果，不因目标/释放者死亡而回溯作废 |
| 7 | 回合结束期 | 1. 锁定本回合定策牌的跨回合持续效果<br>2. 清理临时效果，更新对局状态<br>3. 校验兜底结算触发条件 | 本回合未结算的定策牌效果全部作废，不延续到下回合 |

## 四、同步堆叠式定策牌结算核心规则（全量终版）
本规则为定策牌结算的核心执行标准，操作窗口期关闭后，后台严格按以下堆叠层顺序完成全量预结算，层级顺序永久不可颠倒。

### 4.1 堆叠0层：反制效果结算层
#### 核心结算内容
1.  结算上回合提交的所有反制定策牌，校验触发条件，执行无效化/惩罚效果；
2.  被反制的定策牌，按反制牌描述执行整卡/单效果作废，预扣能量全额返还，不进入后续任何堆叠层；
3.  统计本回合所有有效定策牌（未被反制、未被作废），为后续层级结算提供基准。

#### 无例外执行规则
1.  双方反制牌同步结算，互不干扰，**不可反制其他反制牌**，仅可反制本回合的定策牌；
2.  整卡反制则该卡牌所有效果全部作废，单效果反制不影响同卡牌其他未被反制的效果正常结算；
3.  反制牌仅可针对「单个敌方玩家自身提交顺序的首张定策牌/伤害牌」，禁止设计「敌方阵营全局首张卡牌」类效果，杜绝先后手争议；
4.  本回合提交的反制牌，在本回合堆叠3层完成状态锁定，下回合堆叠0层触发校验，符合「本回合提交、下回合生效」的设计要求；
5.  反制牌锁定后，立即向敌方发送全局预警（仅告知存在反制，不披露卡牌类型、效果、触发条件）。

### 4.2 堆叠1层：防御与数值修正层
#### 核心结算内容
结算所有**不依赖伤害结果的前置效果**，是本回合所有防御、数值变化的最终锁定环节，具体包括：
1.  防御类效果：护甲、护盾、伤害减免、无敌、免伤等防御型定策牌效果；
2.  数值修正类效果：攻击/力量增减、破甲、穿透、伤害加深/降低等直接影响最终伤害数值的定策牌效果；
3.  前置状态类效果：免控、免疫伤害等需提前生效的状态类定策牌效果；
4.  触发式预统计效果：瞬策牌/定策牌中，以「有效定策牌数量/类型」为触发条件的效果，如优化后的「狂怒」，在本层完成最终数值结算。

#### 无例外执行规则
1.  同层内所有效果完全同步结算，无跨阵营/跨玩家先后手；
2.  同玩家自身的效果，按其卡牌锁定的提交顺序依次结算，仅影响自身数值，不破坏公平性；
3.  同名效果规则：同阵营同名增益/减益取最高值，不同阵营效果按全局数值公式线性计算，无强制覆盖；
4.  本层锁定的所有数值，为后续伤害结算的唯一基准，结算完成后不可回溯修改；
5.  预统计触发规则：以「有效定策牌数量/类型」为触发条件的效果，窗口期内仅锁定触发状态，本层完成最终数值结算，窗口期内不对外展示任何变化。

### 4.3 堆叠2层：主动伤害与触发式效果闭环层
#### 核心结算内容
本层为定策牌结算的核心层，分2个原子化步骤封闭执行，全程不跳出本层，不修改之前层级已锁定的结果：
1.  **步骤1：主动伤害一次性同步结算**
    - 所有未被反制的主动定策伤害效果，统一使用堆叠1层锁定的数值，计算最终生效伤害值；
    - 所有伤害牌**不分先后、完全同步、一次性对目标造成伤害，完成血量扣减**；
    - 标记符合触发条件的效果，同步完成濒死状态预标记，暂不执行死亡作废规则。
2.  **步骤2：触发式效果同步闭环结算**
    - 所有预标记的「伤害触发式效果」（反伤、受击回血、击杀触发、伤害叠甲等），**不分阵营、完全同步、一次性结算**；
    - 执行连锁封顶规则：本次结算的衍生效果，无论类型，绝对不再触发新的连锁；
    - 结算完成后，统一执行死亡状态锁定。

#### 无例外执行规则
1.  主动伤害绝对不分次结算，同层内完全同步，可出现双向击杀，不存在「先打死敌方，敌方伤害作废」的先手优势；
2.  触发条件仅以步骤1的主动伤害结算结果为唯一基准，其他层级的效果不可触发连锁；
3.  同一触发条件，单回合仅可触发1次，不可重复触发；
4.  哪怕目标/释放者被预标记死亡，已达成触发条件的效果依然正常结算，不回溯作废；
5.  目标血量扣至0及以下，仅标记死亡状态，本层结算完成后再执行死亡作废规则。

### 4.4 堆叠3层：全局非依赖效果收尾层
#### 核心结算内容
结算所有**不依赖本回合任何结算结果的后置效果**，分2个固定子阶段执行，顺序不可颠倒：
1.  **子阶段1：普通效果结算**
    结算所有未进入前3个层级、未被反制、未被作废的定策牌效果，包括：
    - 控制类效果：沉默、眩晕、减速等（仅影响后续回合，不回溯本回合已提交的操作）；
    - 资源类效果：抽牌、回能、手牌/能量保留等非对抗类资源效果；
    - 支援类效果：临时跨路移动、队友非数值属性加成等支援类效果；
    - 跨回合持续效果：下回合生效的减益/增益效果，本阶段完成状态锁定。
2.  **子阶段2：传说特殊牌专属结算**
    仅结算传说特殊定策牌的效果，优先级永久高于同层普通效果，不被普通效果覆盖。

#### 无例外执行规则
1.  同层内所有效果完全同步结算，无跨阵营/跨玩家先后手；
2.  控制类互斥规则：同目标多控制效果，**持续时间叠加，效果优先级取最高**（全局提前公示：眩晕>沉默>减速>其他）；
3.  增益/减益类规则：同阵营同名效果取最高值，不同阵营效果独立结算，互不覆盖；
4.  同玩家自身的卡牌，按其锁定的提交顺序依次结算；
5.  本回合未生效的跨回合效果，仅锁定状态，不执行结算，下回合开始结算期自动生效。

## 五、专项场景结算闭环规则
### 5.1 多子类型定策牌结算规则
1.  **拆分核心原则**：单张定策牌的多个独立效果，按「效果依赖关系」拆分到对应堆叠层结算，而非整卡强制归属某一子类型；
2.  **生效规则**：拆分后的效果，在对应堆叠层独立结算，互不干扰；某一效果被反制/无效化，不影响同卡牌其他效果的正常结算；
3.  **优先级兜底规则**：若卡牌带「传说特殊子类型」，则该卡牌的所有效果，均进入堆叠3层的传说专属子阶段结算，保证独立优先级；
4.  **设计限制规则**：禁止设计「跨堆叠层反向依赖」的效果（如伤害结算后才生效的破甲、防御效果），避免逻辑混乱。

### 5.2 跨路/分路场景定策牌结算规则
1.  **跨路生效规则**：仅带「跨路生效」标签的定策牌可跨路生效，跨路效果在对应堆叠层同步结算，目标判定以结算时的玩家分路归属为准；
2.  支援场景规则：支援玩家临时移动至目标分路后，其定策牌的目标判定、同路规则，均以目标分路为准，结算完成后自动归位，不影响锁定归属分路；
3.  分路人数限制规则：定策牌效果不可突破「solo路仅1人、双人路最多2人」的强限制，不符合人数规则的支援/换路相关定策牌直接作废；
4.  分路崩塌规则：决战阶段分路合并后，所有定策牌的同路判定覆盖全队3人，跨路标签自动失效。

### 5.3 中枢塔场景定策牌结算规则
1.  **禁用规则**：中枢塔场景内，全程禁用带「跨路生效」标签的定策牌，相关效果直接作废；
2.  小怪清剿阶段规则：定策牌中对敌方玩家生效的部分，结算时完全无效化，仅对PVE小怪/己方玩家生效的部分正常结算，无额外惩罚、不返还能量；
3.  BOSS讨伐阶段规则：定策牌中对敌方玩家生效的部分，结算时完全无效化，仅对BOSS/己方玩家生效的部分正常结算，伤害计入己方阵营总伤害；
4.  状态继承规则：中枢塔内定策牌带来的永久成长效果（卡组优化、遗物、卡牌获取）全程保留，临时效果在中枢塔流程结束后全部清空。

### 5.4 死亡/掉线场景定策牌结算规则
1.  **死亡结算规则**：玩家判定死亡后，本回合已提交、未进入结算的定策牌全部作废，已完成结算的定策牌效果不回溯、不失效；
2.  **临时掉线规则**：玩家临时掉线后，已提交锁定的定策牌正常进入结算流程，未提交的视为放弃本回合操作；
3.  **永久退出规则**：玩家永久退出对局后，已提交锁定的定策牌正常结算，后续回合不再提交任何定策牌操作；
4.  **重连规则**：玩家重连后，直接展示当前回合已完成的结算结果，不重播过往动画，不修改已锁定的结算结果。

## 六、配套动画表现全量执行规范
本规范100%适配同步堆叠式结算规则，严格遵循「先结算后表现」的核心原则，彻底解决结算逻辑与动画表现的冲突。

### 6.1 动画核心执行原则
1.  **预结算前置原则**：所有动画播放前，后台必须完成全量预结算，锁定最终结果，动画仅为结果的可视化展示，绝对不影响结算逻辑；
2.  **单卡连贯原则**：单张定策牌仅做1次完整出场动画，多效果通过「动画时间节点绑定特效」实现，玩家感知不到底层拆分结算，认知零断裂；
3.  **公平性优先原则**：同堆叠层的定策牌动画必须同步播放，无敌我先后手；操作窗口期内绝对不播放任何定策牌动画，不泄露任何操作信息；
4.  **时长控制原则**：单张定策牌完整动画时长严格控制在1.5s以内，单回合全量动画总时长不超过8s，必须设置全局一键跳过动画功能，保障单局8-15分钟的时长要求。

### 6.2 动画与结算绑定的完整时序流程
| 时序 | 后台结算逻辑 | 前台动画表现（玩家可见） | 时长控制 |
|------|--------------|----------------------------|----------|

### 6.3 单卡牌多效果动画实现规范
以「铁斩波（获得4护甲+造成5点伤害）」为例，核心实现标准：
1.  单卡牌仅1次完整出场动画：卡牌从手牌区飞出，居中展示原画+名称，播放「举刀前摇→挥砍攻击」的连贯动作，总时长≤1.5s；
2.  多效果特效绑定时间节点：
    - 前摇动画第0.3s：绑定护甲特效，弹出「+4护甲」数字提示，对应堆叠1层的防御效果；
    - 挥砍命中帧：绑定伤害特效，弹出敌方「-5血量」数字提示，对应堆叠2层的伤害效果；
3.  全程动作连贯无断裂，玩家感知为「一张卡牌打出，同时触发护甲和伤害效果」，完全匹配底层拆分结算逻辑。

### 6.4 特殊卡牌动画处理标准
| 卡牌类型 | 核心动画处理规则 |
|----------|------------------|
| 多子类型定策牌 | 单张卡牌仅1次出场动画，不同效果的特效绑定在动画的对应时间节点，与堆叠层结算顺序对应 |
| 跨回合减益定策牌 | 本回合仅播放核心效果动画，跨回合效果仅挂debuff图标提示，下回合生效时仅弹出极简提示 |
| 反制定策牌 | 本回合仅播放背面出场动画+敌方预警，不泄露卡牌信息；触发回合翻面展示原画，播放对应反制特效 |
| 触发式效果定策牌 | 不做单独出场动画，特效绑定在触发源卡牌的动画节点上，同步播放 |
| 传说特殊定策牌 | 做独立全屏出场动画，放在本回合所有普通卡牌动画结束后播放，搭配永久常驻buff特效 |
| 被反制作废的定策牌 | 仅播放极简的「卡牌碎裂」特效，不播放对应效果动画，不泄露卡牌完整效果 |

### 8.2 核心逻辑校验清单
开发完成后，必须通过以下校验，方可进入测试阶段：
1.  同层级内，双方卡牌同时结算，可出现双向击杀，无先手优势；
2.  多子类型卡牌拆分结算正常，单效果被反制不影响其他效果；
3.  触发式效果连锁封顶正常，无无限循环逻辑漏洞；
4.  窗口期内无任何操作信息泄露，敌方无法通过任何状态变化反推操作；
5.  动画播放不影响结算结果，跳过动画与完整播放的结果100%一致；
6.  所有场景均不突破全局不可突破底层铁律。
